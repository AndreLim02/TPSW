HEADINGS (elementi block)

TITOLO 1 - 32 pixel - titolo pagina: dovrebbe essere UNO SOLO in una pagina

TITOLO 2 - 24 pixel - titolo di secondo livello: se è IMMEDIATAMENTE successivo a un h1, è considerato iul suo sottotitolo, altrimenti è considerato il titolo di un contenuto

TITOLO 3 - 20 pixel - titolo di terzo livello: dimensione ambigua, non utilizzato frequentemente, generalmente considerato sottotitolo dell'h2

TITOLO 4 - 16 pixel - titolo di quarto livello: dimensione predefinita del testo ma in grassetto, utilizzato per il contenuto testuale, generalmente stilizzato senza grassetto

TITOLO 5 - 12 pixel - titolo di quinto livello, generalmente usato per descrizioni/approfondimenti di singoli elementi
TITOLO 6 - 10 pixel - titolo di sesto livello, generalmente utilizzato per contenuti secondari o non importanti

P: paragraph, utilizzato per i BLOCCHI di contenuto testuale, essendo un tag di tipo block
